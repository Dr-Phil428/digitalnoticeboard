# Digital Noticeboard

## Problem
Important information was shared through physical noticeboards. Many students and parents couldn’t access them in time leading to missed updates.

## Solution
I built a **Digital Noticeboard** that delivers notices online and it ensures:
- Instant updates
- Easy access from the available access points around the school
- Organized information

## Impact
The project has:
- **Improved Accessibility:** Everyone can view notices anytime.
- **Enhanced Communication:** Updates reach the audience quickly and reliably.
